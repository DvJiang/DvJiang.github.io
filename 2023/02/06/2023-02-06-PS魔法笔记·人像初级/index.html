这篇文章用于记录我学习了1个小时PS的成果。

不得不说，PS被称为四大神术之一是有道理的。只需要稍微学一下，就可以让人改头换面，P得妈都认不出来。

快捷键（MAC版）

放大缩小：option配合鼠标滚轮

笔的大小：英文模式下按中括号键

复制图层：command + J

合并图层（并保留原有图层）： command + option + shift + E

导出： command + option + shift + W

处理技巧/一般流程

暗区不规则亮色：

滤镜 - 杂色 - 减少杂色

不过大部分时候都不需要这个

DR5：DB处理

让人脸显得亮一些，加白的同时可以去除眼袋和法令纹，注意控制笔的流量，10就已经很大了

DR5：智能修图磨面

用椭圆形框套住脸，生成一个磨皮的蒙版，然后把痘痘之类的都可以除去。

液化

主要用来控制人的身材和五官的。

首先用人脸识别调整人的五官，建议按照三庭五眼的规则调整，眼睛可以调大一些。

然后用变形工具控制一下假发高度，腰的宽度和腿长。

最后用恢复工具把明显的形变的地方改回去，让它看起来自然些。

图章/污点修复

用来除掉一些不该存在的小东西，比如身上的痣，别针等。

蒙版调色

按Q打开蒙版，然后注意笔的颜色正确，流量为100%，然后涂画所有除了脸以外的皮肤，再按Q，就可以圈出这些地方。

然后在属性里打开「可选颜色」，注意蒙版可能需要反转，然后把黄色和黑色调低，青色和红色跳高，让皮肤更白更鲜红。
